Alpha

Continuously Responsive Website


This website will smoothly and continuously respond as the window size narrows using a combination of CSS flex and grids for a design that looks right on different screens.

Each element including the containers and fonts can resize within sensible limits to accomodate screens from 300px upwards with no need for side scroll.

Built with GitHub, GitPod, VSCode, HMTL & CSS.

Modular Design


Shared Card classes mean expanding the page is simple and styling is consistent.

Quick Loading


Using CSS and HTML without JS for faster loading times.

No Media Queries


Responsiveness has been designed to remove the need for media queries.
