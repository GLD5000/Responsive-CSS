H1 Hello, I am G

Project

A continuously responsive website using CSS and HTML only.

The website will smoothly and continuously respond as the window size narrows using a combination of CSS flex and grids for a design that looks right on different screens.

Each element including the containers and fonts can resize within sensible limits to accomodate screens from 300px upwards with no need for side scroll.

Modular design


Each card is its own entity and classes are arranged so the more cards can be added with consitent styling.

Quick loading


Moderate usage of images coupled with pure CSS and preloaded webfonts make for fluid and fast load times.

No media queries required


Responsiveness has been designed to remove the need for media queries.
